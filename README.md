# Studio Journal

Studio Journal is a modern editorial blog template built with HTML, CSS, and JavaScript. It was redesigned from an earlier class project into a polished front-end portfolio piece that feels more aligned with real client work.

## Live Demo

[View Live Site](https://kiwichamellejones.github.io/studio-journal/)

## Overview

This project showcases a clean editorial layout for content-driven brands, personal blogs, lifestyle websites, or online magazines. The goal was to create a responsive and visually elevated reading experience using only core front-end technologies.

## Features

- Responsive editorial-style homepage
- Modern hero section with call-to-action buttons
- Featured story section
- Article card layout with images and category labels
- JavaScript-powered category filtering
- Mobile navigation menu
- Newsletter sign-up section
- Clean, polished UI with strong typography and spacing

## Built With

- HTML5
- CSS3
- JavaScript

## Project Purpose

This project was created as a portfolio-ready redesign of a coding class assignment. The original version was a niche family-history blog, and this updated version repositions it as a more professional, reusable editorial website template.
